# Aplikasi Kelulusan SMK

Sistem informasi berbasis web untuk pengumuman kelulusan SMK — dibangun dengan **Next.js**, **Supabase**, dan di-deploy ke **Vercel**.

## Tech Stack

- **Frontend & Backend:** Next.js (App Router) + Tailwind CSS + shadcn/ui
- **Database & Auth:** Supabase (PostgreSQL, Auth, Storage, Realtime, RLS)
- **Deployment:** Vercel
- **PDF:** @react-pdf/renderer / Puppeteer
- **Notifikasi:** Nodemailer (email) + Fonnte (WhatsApp)
